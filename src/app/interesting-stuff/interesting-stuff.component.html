Sharplab

I am currently part of an exciting project directed by the McGill Sharplab.

With the advent of covid, psychological testing has been heavily impacted as it has become difficult to safely invite and test participants for studies.

This project aims to build a web application that can administer psychological tests over the internet through web based technologies.
